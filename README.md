# 텍스트 어드벤쳐
"살아있는 세계, 움직이는 사람들, 그리고 당신만의 선택으로 심연의 깊은 곳으로 내려가세요.
각 계층마다 다른 위험과 규칙이 기다립니다. 자원을 관리하고, 탐험가들과 관계를 쌓으며, 때로는 배신하고 때로는 협력하세요. 돌아갈 것인가, 더 깊이 내려갈 것인가 - 매 순간이 선택입니다.
탐험가들은 각자의 목표를 가지고 움직이며, 당신의 행동에 반응합니다. 정해진 스토리는 없습니다. 당신이 만드는 이야기가 펼쳐집니다.
텍스트 기반 시뮬레이션 어드벤처 - 계층마다 다른 세계, 매번 다른 여정"
